# animal-status
Serverless API for maintaining state of shelter animals

# Getting started

## One-time setup
* Install Node.js LTS
* Install the AWS CLI tools
* ```npm install serverless -g```
* ```aws configure```

## Deploying changes
* ```serverless deploy```
